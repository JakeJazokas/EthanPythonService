To install the service run:
	- python EthanFitnessService.py install

To remove the service run (After it is stopped):
	- python EthanFitnessService.py remove

After the service is installed, run the following command to start it:
	- net start EthanFitnessService

To stop the service, run the following command:
	- net stop EthanFitnessService
